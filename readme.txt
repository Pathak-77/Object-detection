---Setup Guide for YOLO Object Detection

To get started with YOLO object detection, follow these steps:

---Clone the Repository:
   - Clone or fork the YOLO repository from [Ultralytics GitHub]   ->  (https://github.com/ultralytics/ultralytics) or directly clone it using: 
    
     git clone https://github.com/ultralytics/ultralytics.git
     

---Install Required Libraries:


     - `torch` (version 2.3.1)
     - `torchvision` (version 0.18.1)
     - `ultralytics` (version 8.2.87)
     - `opencv-python` (version 4.10.0.84)
     - `opencv-contrib-python` (version 4.10.0.84)
     - `numpy` (version 1.26.4)
     - `pillow` (version 10.4.0)
     - `matplotlib` (version 3.9.2)
     - `scipy` (version 1.14.1)
     - `requests` (version 2.32.3)

--- You can install these libraries using pip:
    
     pip install torch==2.3.1 torchvision==0.18.1 ultralytics==8.2.87 opencv-python==4.10.0.84 opencv-contrib-python==4.10.0.84 numpy==1.26.4 pillow==10.4.0 matplotlib==3.9.2 scipy==1.14.1 requests==2.32.3
